Manual de Uso Científico

1. Autenticación

Para acceder, use un correo que termine exactamente en @juventud.edu.mx.

2. Ingreso de Fórmulas (Pegado Directo)

La pantalla LCD de la calculadora es interactiva. Puede hacer clic directamente en ella y escribir desde su teclado físico, o usar Ctrl+V (Pegar) para insertar fórmulas complejas copiadas de otras fuentes.

3. Modos de Visualización

  • 1D: Recta numérica para evaluar constantes (ej. 5 * a).
  • 2D: Plano cartesiano (variables x, y).
  • 3D: Superficie espacial (variables x, y, z). Incluye ejemplos como la Esfera.

4. Funciones Científicas y Exponentes

Use el botón 2nd para alternar entre funciones trigonométricas directas e inversas. El botón RAD alterna entre radianes y grados (DEG).

  • Exponentes: Use eˣ para exponencial natural, 10ˣ para base 10, y EXP para notación científica (*10^n).
  • Memoria: MC (Limpiar), MR (Recuperar), M+ (Sumar), M- (Restar).
  • Combinatoria: nPr(n, r) para permutaciones, nCr(n, r) para combinaciones.
  • Factorial: Use ! después de un número (ej. 5!).

5. Múltiples Gráficas

Puedes graficar hasta 10 funciones simultáneamente. Cada una puede tener un color distinto.

  • Presiona ENTER para compilar la ecuación actual.
  • Presiona SGT para fijar la gráfica y editar la siguiente automáticamente.
  • Use el menú desplegable para cambiar entre los 10 espacios de gráficas.

6. Sintaxis Avanzada

El motor soporta multiplicación implícita (2x = 2*x).

  • Ecuaciones Implícitas: x^2 + y^2 = 9 (Dibuja un círculo).
  • Esfera 3D: x^2 + y^2 + z^2 = b^2 (Ajuste el radio con el parámetro B).
  • Constantes: pi (π), e, phi (Proporción áurea).
  • Animación: Incluya la variable t y presione ▶️.
